# workpeopleProject
 
 

코로나19 장기화로 재택근무가 증가함에 따라 집에서도 쉽게 업무를 볼 수 있는 그룹웨어가 필요하다고 느꼇으며, 

그룹웨어을 사용한 재택근무루 업무 효율성이 높아지고, 개인의 시간을 아낄 수 있다는 장점이 있어  누구나 쉽고 간편하게 사용할 수 있는 그룹웨어를 개발하였습니다.

맡은 파트는 인사관리,조직도,주소록 파트가 있습니다.
